# Superstore Sales Data Analysis

## Project Overview

In this project, I performed exploratory data analysis on the Superstore Sales dataset to understand sales performance, profitability, and regional trends.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Analysis

The analysis focused on understanding key factors that influence sales and profit.

The following visualizations were created:

* Sales by product category
* Profit by category
* Sales by region
* Top products based on sales
* Sales distribution across categories

## Key Insights

* Technology category generates the highest sales compared to other categories.
* Profitability varies across categories and regions.
* A small number of products contribute significantly to total sales.
* Regional differences impact overall sales performance.

## Conclusion

The analysis highlights important sales patterns and helps identify key categories and products driving revenue.
